Usage Example

Basic usage:

const cursor = new AuraCursor({ size: 20, color: '#000000' });

cursor.init();

With hover effect options:

const cursor = new AuraCursor({

  size: 20,

  color: '#000000',

  hoverEffect: {

    color: '#ff6b6b',

    opacity: 0.8,

    scale: 1.5

  }

});

cursor.init();

With outline mode:

const cursor = new AuraCursor({

  size: 20,

  color: '#000000',

  opacity: 0.5,

  outlineMode: true,

  outlineWidth: 2,

  centerDotColor: '#ff0000',

  centerDotHoverColor: '#ff4fa2',

  centerDotSize: 5

});

cursor.init();

Update options:

cursor.updateOptions({ size: 40, color: '#ff0000' });

Destroy:

cursor.destroy();
